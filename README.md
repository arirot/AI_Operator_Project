# AI Operator Assistant with XAI in a Digital Twin (Production Line)

This project builds an explainable AI assistant for a production-line digital twin.  
It predicts machine failure risk, explains decisions using SHAP + LIME, and provides a what-if simulation dashboard for operators.

---

## Features
- Predictive maintenance model (XGBoost)
- SHAP global + local explanations
- LIME rule-based explanations
- Interactive Streamlit operator dashboard
- What-if analysis for operator decision support
